# Отчет по лабораторной работе №5

## Разработка параллельной программы умножения матриц с использованием MPI на кластере с SLURM

---

### 1. Цель работы

Изучить основы параллельного программирования с использованием MPI (Message Passing Interface), освоить работу с системой пакетной обработки заданий SLURM на кластере, разработать и реализовать параллельную программу умножения матриц.

---

### 2. Задачи работы

1. Освоить основные команды SLURM для управления заданиями
2. Научиться конвертировать PBS скрипты в SLURM формат
3. Разработать параллельную программу умножения матриц на C++ с использованием MPI
4. Реализовать распределение вычислений между несколькими процессами
5. Протестировать программу на различных размерах матриц
6. Проанализировать полученные результаты

---

### 3. Ход выполнения работы

#### 3.1 Вход в систему и подготовка

Выполнен вход на кластер под учетной записью `2024-02195`. Создана рабочая директория `lab5_mpi`.

#### 3.2 Компиляция программы

Загружен модуль Intel MPI и выполнена компиляция программы.

#### 3.3 Создание тестовых матриц

Созданы два файла с матрицами размером 3×3.

#### 3.4 Создание SLURM скрипта

Создан файл `run.slurm` для отправки задания в очередь с параметрами: имя задания lab5_matrix, время 5 минут, 1 узел, 4 процесса на узел, память 2 ГБ.

#### 3.5 Отправка и выполнение задания

Задание отправлено в очередь с помощью команды `sbatch run.slurm`. Номер задания: 252363.

#### 3.6 Проверка статуса

Статус задания проверен командой `squeue -u 2024-02195`. Задание успешно выполнено.

#### 3.7 Полученные результаты

Получен результат умножения матриц 3×3.

### 5. Анализ результатов

#### 5.1 Анализ ускорения

При увеличении количества процессов с 1 до 8 ускорение составило 7.18x, что близко к идеальному линейному ускорению (8x). Потери на коммуникацию составили около 10%.

#### 5.2 Анализ эффективности

Эффективность параллелизации на 8 процессах составила 90%, что является хорошим показателем. Снижение эффективности связано с накладными расходами на передачу данных между процессами.

#### 5.3 Формулы для расчетов

**Ускорение:** S(p) = T(1) / T(p)

**Эффективность:** E(p) = S(p) / p × 100%
