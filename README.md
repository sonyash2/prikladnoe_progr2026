# Отчет по лабораторной работе: Параллельное умножение матриц с использованием MPI

## 1. Цель работы
Модифицировать программу из лабораторной работы №1 для параллельной работы по технологии MPI. Исследовать эффективность параллельного умножения квадратных матриц.

## 2. Ход выполнения работы
Разработана программа на C++ с использованием MPI. Проведено тестирование на матрицах размером 200, 400, 800, 1200, 1600, 2000 с использованием 1, 2, 4 и 8 процессов.
