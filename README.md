Лабораторная работа - Умножение матриц

Список файлов в проекте

main.cpp Исходный код программы на C++. Содержит класс MatrixMultiplier для умножения матриц, поддержку многопоточности, измерение времени

verify.py Скрипт на Python для верификации результатов. Использует библиотеку NumPy для проверки правильности умножения

verification_data.txt Файл с данными для проверки: матрицы A, B и результат C, созданный программой

results.csv Результаты эксперимента: размер матрицы, потоки, время, операции, MFLOPS

report.md Полный отчет по лабораторной работе с описанием задания, реализации, результатов и выводов

README.md Краткое описание всех файлов проекта
Как запустить

1. Скомпилировать C++ программу
2. Запустить main.cpp
3. Ввести данные
4. Запустить verify.py для проверки

Краткое содержание

Программа умножает квадратные матрицы параллельно и последовательно. Проверено на матрице 2x2. Результаты совпадают с NumPy. Время выполнения: 0.005 секунд.
